# Common Club Management System Backend

A single application to manage activities of all clubs in an educational       institution.

## Tech Stack

- Expres
- Typescript
- PostgreSQL
- Prisma

## Steps to run the code

``` 
  git clone https://github.com/Priyanshu-agr/ccms.git

  cd ccms
  
  npm i

  npx prisma init

  npx tsc --init

  npm run dev

```
